# SAM 3 標註工具 - 完整使用手冊

## 給初學者的話

歡迎使用 SAM 3 標註工具！這是一個幫助你在圖片上標記物件的軟體。

**什麼是「標註」？**
想像你在照片上用螢光筆圈出重要的東西，並寫上它是什麼。這就是標註！電腦需要這些標註來學習辨識物件。

**這個工具能做什麼？**
- 在圖片上框選或點擊物件
- 自動辨識物件的邊界（使用 AI 技術）
- 儲存標註資料供電腦學習使用

## 第二章：開始使用（快速入門）

### 步驟 1：準備圖片

首先，把你要標註的圖片放到一個資料夾裡。

**建議做法：**
1. 在 D 槽建立一個資料夾，例如 `D:\待標註圖片`
2. 把所有圖片複製進去
3. 圖片格式可以是 `.jpg`、`.png`、`.jpeg`、`.bmp`

```
D:\待標註圖片\
├── photo001.jpg
├── photo002.jpg
├── photo003.jpg
└── ...
```

### 步驟 2：啟動程式

打開命令提示字元（按 Win+R，輸入 `cmd`，按 Enter）

輸入以下指令：
```
cd sam3_labeler_package
python sam3_labeler.py
```

等待幾秒鐘，瀏覽器會自動開啟。

**如果沒有自動開啟：** 手動打開瀏覽器，輸入 `http://localhost:7860`

### 步驟 3：載入圖片

1. 在「圖片資料夾」欄位輸入你的圖片路徑
   ```
   D:\待標註圖片
   ```

2. 在「輸出資料夾」欄位輸入你要存放結果的路徑
   ```
   D:\標註結果
   ```

3. 點擊「載入」按鈕

4. 成功的話，你會看到第一張圖片顯示出來！

### 步驟 4：開始標註

最簡單的方法是使用「點擊分割」：

1. 在左側工具面板選擇「點擊分割」
2. 在右側確認「目前類別」是你要標的物件類別
3. 用滑鼠點擊圖片上的物件中心
4. AI 會自動辨識並框出物件！

**看到彩色框框出現就表示成功了！**

### 步驟 5：切換到下一張

點擊圖片右邊的 → 箭頭，或按鍵盤的 → 方向鍵

**重要：切換圖片時，標註會自動儲存！**

## 第三章：三種標註方法詳解

### 方法一：點擊分割（最簡單）

**適合情況：** 物件明顯、邊界清楚

**操作步驟：**
1. 選擇工具「點擊分割」
2. 選擇正確的類別
3. 點擊物件的中心位置
4. 完成！

```
    點這裡
      ↓
  ┌───────┐
  │   ●   │  ← 點擊物件中心
  │ 瓶子  │
  └───────┘
```

**小技巧：**
- 盡量點在物件的正中央
- 如果效果不好，可以點其他位置再試一次

### 方法二：矩形框選（較精確）

**適合情況：** 物件重疊、AI 判斷錯誤時

**操作步驟：**
1. 選擇工具「矩形框選」
2. 選擇正確的類別
3. 點擊物件的左上角 → 畫面會顯示一個標記
4. 點擊物件的右下角 → AI 開始辨識

```
  第一次點這裡
      ↓
      ●─────────┐
      │         │
      │  物件   │
      │         │
      └─────────●
                ↑
          第二次點這裡
```

**小技巧：**
- 框選範圍要包含整個物件
- 不要框太大，盡量貼近物件邊緣

### 方法三：文字分割（自動找物件）

**適合情況：** 想要一次找出圖片中所有特定物件

**操作步驟：**
1. 在「文字提示」欄位輸入物件名稱
   - 單一物件：`bottle`
   - 多個物件：`bottle, debris, buoy`（用逗號分開）
2. 點擊「執行文字分割」
3. AI 會自動找出所有符合的物件！

**注意：** 輸入的文字會自動變成新的類別

## 第四章：編輯已有的標註

### 4.1 選取標註

1. 切換到「選取標註」工具
2. 點擊圖片上的標註框
3. 被選中的標註會變成黃色粗框

**多選方法：**
- 一個一個點：每點一個就加入選取
- 框選多個：在空白處點一下，然後點另一個位置，中間的標註都會被選中

### 4.2 刪除標註

1. 選取要刪除的標註（可以多選）
2. 點擊右側的「刪除選取」按鈕
3. 標註消失了！

### 4.3 修改類別

假設你不小心把「瓶子」標成「垃圾」，想要改正：

1. 選取那個標註
2. 在右側「類別」下拉選單選擇正確的類別
3. 點擊「改變類別」按鈕
4. 完成修正！

### 4.4 取消選取

如果選錯了，點擊「取消選取」按鈕，所有選取會被清除。

## 第五章：類別管理

### 什麼是類別？

類別就是物件的種類。例如：
- `debris` = 碎屑/垃圾
- `bottle` = 瓶子
- `buoy` = 浮標
- `boat` = 船

### 新增類別

**方法一：手動新增**
1. 在右側「新增類別」欄位輸入名稱
2. 點擊「新增」按鈕

**方法二：自動新增**
使用「文字分割」功能時，輸入的名稱會自動變成類別

### 切換類別

每次標註前，記得先選對類別！

**快速切換：** 點擊右側的 Radio 按鈕
**完整選單：** 使用下拉選單

## 第六章：儲存與輸出

### 自動儲存

好消息！這個程式會自動儲存你的標註：
- 切換到下一張圖片時 → 自動儲存
- 切換到上一張圖片時 → 自動儲存

**你不需要手動按儲存！**

### 手動儲存

如果你想立即儲存，點擊右側的「手動儲存」按鈕。

### 輸出的檔案

程式會產生以下檔案：

```
D:\標註結果\
├── images\              ← 已標註的圖片副本
│   ├── photo001.jpg
│   └── photo002.jpg
│
├── labels\              ← OBB 格式標註檔
│   ├── photo001.txt
│   └── photo002.txt
│
├── labels_seg\          ← 多邊形格式標註檔
│   ├── photo001.txt
│   └── photo002.txt
│
├── masks\               ← 遮罩圖像（如果有開啟）
│   ├── photo001.png
│   └── photo002.png
│
```

### 輸出格式說明

| 格式 | 檔案位置 | 用途 |
|------|----------|------|
| OBB | labels/*.txt | 用於訓練 YOLO OBB 模型（旋轉框偵測） |
| Seg | labels_seg/*.txt | 用於訓練 YOLO-Seg 模型（實例分割） |
| Mask | masks/*.png | 用於語意分割或製作遮罩 |

**給初學者的建議：** 保持預設的 OBB 和 Seg 打勾即可！

## 第九章：常見問題 Q&A

### Q1：點擊物件但沒有反應？

**可能原因：**
- 工具選錯了 → 確認選的是「點擊分割」
- AI 無法辨識 → 改用「矩形框選」試試

**解決方法：**
1. 勾選右側的「SAM 失敗改用方框」
2. 這樣即使 AI 失敗，也會用你框的範圍建立標註

### Q2：標註的形狀不對？

**解決方法：**
1. 刪除錯誤的標註
2. 改用「矩形框選」，框選範圍更精確
3. 或者多試幾次「點擊分割」，點不同位置

### Q3：忘記選類別就標註了？

**解決方法：**
1. 選取那個標註
2. 選擇正確的類別
3. 點「改變類別」

### Q4：圖片太大看不清楚？

**解決方法：**
- 使用瀏覽器的縮放功能（Ctrl + 滾輪）
- 或按 Ctrl + +/- 放大縮小

### Q5：程式當掉了，標註會不見嗎？

**不用擔心！**
- 只要你有切換過圖片，之前的標註都已經儲存了
- 重新啟動程式，載入相同資料夾即可繼續

### Q6：如何備份我的標註？

直接複製整個輸出資料夾（例如 `D:\標註結果`）到其他地方即可。

### Q7：可以標註影片嗎？

目前不支援直接標註影片。你需要先把影片轉成圖片序列。

## 第十一章：標註的好習慣

### 1. 標註要完整
確保物件的邊界都在框框內，不要漏掉一部分。

```
  ✓ 正確           ✗ 錯誤
┌─────────┐      ┌──────┐
│  ████   │      │  ████│←── 物件被切掉了
│  ████   │      │  ████│
└─────────┘      └──────┘
```

### 2. 標註要緊密
框框盡量貼合物件，不要留太多空白。

```
  ✓ 正確           ✗ 錯誤
┌─────┐          ┌───────────┐
│ ███ │          │           │
│ ███ │          │   ███     │ ←── 太多空白
└─────┘          │   ███     │
                 └───────────┘
```

### 3. 每個物件單獨標
一個標註只框一個物件，不要把多個物件框在一起。

```
  ✓ 正確              ✗ 錯誤
┌───┐  ┌───┐       ┌─────────┐
│ A │  │ B │       │  A   B  │ ←── 兩個物件框在一起
└───┘  └───┘       └─────────┘
```

### 4. 類別要一致
相同的物件永遠用相同的類別名稱。

```
✓ 正確：bottle, bottle, bottle
✗ 錯誤：bottle, Bottle, 瓶子  ←── 這會被當成三種不同類別！
```

### 5. 定期備份
每標註 50-100 張圖片，就備份一次輸出資料夾。

## 第十二章：名詞解釋

| 名詞 | 解釋 |
|------|------|
| **標註 (Annotation)** | 在圖片上標記物件位置和類別的動作 |
| **類別 (Class)** | 物件的種類，例如「瓶子」、「船」 |
| **OBB** | Oriented Bounding Box，旋轉邊界框，可以斜的方框 |
| **Seg** | Segmentation，分割，指物件的精確邊界 |
| **Mask** | 遮罩，黑白圖像表示物件位置 |
| **SAM** | Segment Anything Model，Meta 開發的 AI 分割模型 |
| **AI/人工智慧** | 讓電腦自動辨識物件的技術 |
| **GPU** | 顯示卡，用來加速 AI 運算 |

## 第十三章：完整操作範例

### 範例：標註海漂垃圾圖片

**情境：** 你有 100 張海邊的照片，要標註照片中的垃圾、瓶子、浮標。

**步驟：**

1. **準備工作**
   - 把 100 張圖片放到 `D:\海邊照片`
   - 建立輸出資料夾 `D:\海邊標註`

2. **啟動程式**
   ```
   cd sam3_labeler_package
   python sam3_labeler.py
   ```

3. **載入圖片**
   - 圖片資料夾：`D:\海邊照片`
   - 輸出資料夾：`D:\海邊標註`
   - 點擊「載入」

4. **設定類別**
   - 新增類別：`debris`（碎屑）
   - 新增類別：`bottle`（瓶子）
   - 新增類別：`buoy`（浮標）

5. **開始標註第一張圖**
   - 看到一個瓶子 → 選擇類別 `bottle` → 點擊瓶子中心
   - 看到一些碎屑 → 選擇類別 `debris` → 點擊碎屑
   - 重複直到圖片中的物件都標完

6. **切換到下一張**
   - 點擊 → 箭頭
   - 狀態列顯示「已自動儲存」

7. **發現標錯了**
   - 切換到「選取標註」工具
   - 點擊標錯的標註
   - 點擊「刪除選取」
   - 重新標註

8. **繼續標註...**
   - 重複步驟 5-7
   - 每標 10 張休息一下眼睛！
